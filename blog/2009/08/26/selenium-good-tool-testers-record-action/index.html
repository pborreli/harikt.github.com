Selenium a Good Tool for Testers to Record the Action

Aug 26th, 2009

Learning Zend framework made me explore many new tools . The selenium IDE which is an add on for Mozilla is a wonderful tool for testers . For eg : we are humans and we may forget the actions done what all things we do . But we can record our clicks and inputs using selenium IDE . So when an error occurs we can show the developer what happens and what was the input entered etc . I am really excited to see the Selenium IDE .
